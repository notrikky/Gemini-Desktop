# Gemini Desktop App

A fast, unofficial, and extremely clean desktop wrapper for Google Gemini, built with Electron.

This app brings Google's AI directly to your desktop—packaged in a sleek window that integrates seamlessly into your system.

## Features
* **System Tray Integration:** The app runs resource-friendly in the background and is instantly accessible via an icon next to your Windows clock.
* **Invisible Autostart:** Boots directly with Windows in the background without throwing an annoying window in your face.
* **Single-Instance Lock:** Prevents accidental opening of multiple app clones.
* **Seamless Login:** Uses a custom user-agent to smoothly bypass Google's "Insecure Browser" blocks.

## 🚀 Download & Installation

Not a developer and just want to use the app? 
Head over to the **[Releases]** section on the right and download the latest version:

1. **Setup.exe:** Installs the app, sets up the autostart, and adds the logo to your start menu.
2. **Portable .zip:** Just extract the folder and run the `.exe`. No installation required!
